# Testimonials-Manager
An encapsulated Zen Cart plugin, based on v3.0 authored by davewest.  Enables a site to accept and display testimonials from customers.

**Zen Cart download link:** [https://www.zen-cart.com/downloads.php?do=file&id=2369](https://www.zen-cart.com/downloads.php?do=file&id=2369)

**Zen Cart support thread:** [https://www.zen-cart.com/showthread.php/229376-Testimonials-Manager-Support-Thread](https://www.zen-cart.com/showthread.php/229376-Testimonials-Manager-Support-Thread)

Refer to the [readme.html](https://htmlpreview.github.io/?https://github.com/lat9/Testimonials-Manager/blob/main/readme.html) and the [Wiki](https://github.com/lat9/Testimonials-Manager/wiki) in this GitHub repository for additional details.
