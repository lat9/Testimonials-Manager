ABOUT Testimonials Manager
==========================
This module lets my customers leave Feedback for me on how I performed and how my site worked for them.  As with product reviews we use a star rating system, voting system for each feedback. 

Avatars are user images commonly used around many sites that use customer feedback. I for one know that url images could lead to issues thus I use in-house images and modified uploaded images.  User uploads are resized, re-coded, and striped of any identifying information such as location latitude longitude information, and micro viruses.

Installation Instructions for zc-1.5.8 and zc-1.5.7d only
========================================================================
(1 * Fisrt drop all folders and files in BOTH-zc15x
(2 * with Zen Cart 1.5.7d, drop all folders and files in ONLY-zc157
(2 * with Zen Cart 1.5.8x, drop all folders and files in ONLY-zc158
========================================================================
You should not have any problems.. You can drag and drop most of it, there are some core file edits for links and meta-data fields. 

Move all files over with the exception of information sidebox, meta-data and account files, they should be compared with existing files.

Testimonials template files and admin was used SVG fonts with FontAwesome.  I use ONLY SVG's without fonts!
