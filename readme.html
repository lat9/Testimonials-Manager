Purpose

This (now encapsulated) version of the plugin lets customers leave feedback as to how the site performed and how it worked for them. As with product reviews, a star rating is used for the testimonials' ranking.

The plugin is based on the like-named Testimonials Manager plugin by @davewest, last published in 2023. The support for customer avatars has, however, been removed from the current version.

This version has been tested with Zen Cart v2.2.2 and the current Zen Cart Github repository, using both the responsive_classic and ZCA Bootstrap templates.

Installing, Upgrading or Uninstalling

These sections identify the processes you'll use to initially install Testimonials Manager, upgrade or uninstall the plugin entirely.

Note: If you want to display and/or edit the define-page for the testimonials_add page, you must ensure that you have applied the Zen Cart core-file changes provided in this Zen Cart GitHub PR. If those changes aren't applied, you won't be able to edit the define-page via the site's admin and it's possible that the testimonials_add page will whitescreen with a PHP error on the storefront.

Initial Installation or Upgrade

Note: If you are upgrading from a previous, non-encapsulated, version of the plugin, you will need to remove all files provided by that previous version prior to installing an encapsulated version. Failure to do so will most likely result in fatal PHP errors (both admin and storefront). You can keep your configuration settings and they will be upgraded automatically.

Start by copying the Testimonials Manager distribution's zc_plugins directory (and all its files) to the root of your site's file-system. You should see a zc_plugins/TestimonialsManager sub-directory when completed and …

  1. Login to your Zen Cart's admin
  2. Navigate to Modules :: Plugin Manager.
  3. Click on the row that displays a Plugin Key of TestimonialsManager and click the associated Install button.
  4. All finished! You'll now see:
    • A Testimonial Manager selection under the admin's Configuration tab.
    • A Testimonial Manager selection under the admin's Tools tab.
    • A new sidebox available when viewing the admin's Tools :: Layout Boxes Controller: TestimonialsManager/{version}/testimonials_manager.php.

Uninstalling

To uninstall Testimonials Manager …

  1. Navigate to your Zen Cart admin's Modules :: Plugin Manager.
  2. Click on the row that displays a Plugin Key of TestimonialsManager and click the associated Un-Install button.
  3. From the confirmation form, click that Un-Install button. That button-click will "inactivate" the plugin.
  4. If you also want to remove the files associated with the plugin, click the Clean Up button.
